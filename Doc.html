Tic Tac Toe Game

of 3 players including a computer player

Used Technologies:

    • Java 8
    • Gradle 4.6
    • JUnit 4.12

How to Play:

  • Game has default entry A, B, C for players characters and 5 game board side length L, if you wish to customize these configurations go to config.properties file and change them without changing the file formate
  • After running the game application, the game starting player will be randomly picked by the computer everytime.
  • Every player will be invited to take his move by entering a single line entry in this format x,y replresenting the position he wants to pick where x is the row number from 0 to L-1 and y is the column number from 0 to L-1.
